# 我就想知道，热水器到底是即插即用省电，还是一直插电省电

为了教某个初中生掌握学习的方法，让他研究上述问题。


## 结论 
任何时候都是即插即用省电。 即插即用的热水器热损耗比一直插电的热水器热损耗多。
